
## Usage
```
python 3.6
gym 0.19.0
torch 1.10.2
```

To train from scratch:
```
python main.py
```

To test model:
```
python main.py --mode test
```


## How it works

[main.py](main.py) is our executable. **PPO** policy is here! 

[grid.py](main.py) is our env. **Greedy/Random** policy is here!
